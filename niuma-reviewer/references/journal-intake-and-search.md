# Journal Intake and Search Protocol

Use this protocol before judging journal fit or author-instruction compliance.

## Required Journal Evidence

Collect a compact evidence table with:

| Item | What to capture |
|---|---|
| Journal identity | Full journal name, publisher/society, ISSN if useful |
| Aims and scope | Target disciplines, populations, intervention/topic preferences, excluded content |
| Article type | Original research, review, brief report, viewpoint, protocol, case report, quality improvement, etc. |
| Current author instructions | Submission format, word limits, abstract structure, figures/tables, references, supplementary files |
| Reporting standards | CONSORT, STROBE, PRISMA, TRIPOD, STARD, CARE, CHEERS, SQUIRE, SPIRIT, SRQR/COREQ, ARRIVE, DECIDE-AI, CONSORT-AI, SPIRIT-AI, or journal-specific checklists |
| Ethics policies | IRB/ethics approval, consent, trial registration, data sharing, conflicts, funding, patient privacy |
| Editorial priorities | Novelty threshold, readership, open science expectations, policy/practice implications |
| Source metadata | URL, page title, publication/update date if shown, access date |

## Source Hierarchy

1. Official journal pages: aims/scope, instructions for authors, article types, editorial policies.
2. Official publisher or society policy pages for ethics, reporting, data availability, AI use.
3. EQUATOR Network or official guideline pages for reporting standards.
4. Recent published articles in the same article type from the target journal, used only as exemplars of style and structure.
5. Indexing metadata such as PubMed, Crossref, Scopus, or DOAJ only to resolve identity or article examples.

## Search Pattern

Use targeted queries such as:

- `[Journal Name] aims scope`
- `[Journal Name] instructions for authors`
- `[Journal Name] article types original research abstract`
- `[Journal Name] reporting guidelines ethics data sharing`
- `[Journal Name] [article type] site:journal/publisher-domain`

If several journals have similar names, verify publisher, ISSN, and URL before using a source.

## Journal Fit Questions

Answer these before reviewing details:

- Does the manuscript topic sit inside the journal's stated scope?
- Does the article type match what the journal accepts?
- Is the population, setting, method, or technology relevant to the journal's readership?
- Does the paper meet minimum transparency requirements for ethics, registration, data, funding, and conflicts?
- Are there journal-specific dealbreakers, such as refusal of case reports, mandatory structured abstract, or required checklist upload?

## Evidence Discipline

When writing the review, cite journal requirements with links. Paraphrase rules instead of over-quoting. If a requirement cannot be verified, write "not found in the sources checked" rather than assuming.
