# Journal Requirements Extraction

Use this reference when converting author guidelines into writing actions.

## Source Priority

| Priority | Source | Use for |
|---:|---|---|
| 1 | Official journal instructions for authors | Article type, limits, headings, required statements |
| 2 | Official journal article-type page | Scope and article category fit |
| 3 | Publisher policy page | Ethics, data, code, reporting policies |
| 4 | Submission system notes | Upload and formatting constraints |
| 5 | Exemplar articles | Actual writing habits only, not official rules |

If a detail is unstable, current, or journal-specific, verify online before final advice.

## Requirements Matrix

Extract only items that change the manuscript:

| Field | What to record | Writing implication |
|---|---|---|
| Article type | Name, eligibility, scope | Decide whether the manuscript fits |
| Abstract | Structured/unstructured, headings, word limit | Rewrite abstract to match |
| Main text | Word limit, required headings | Cut or expand sections |
| Methods | Required details and checklists | Add transparent reporting language |
| Ethics | Approval, consent, registration | Add required statements or placeholders |
| Data/code | Availability, repositories, supplementary data | Add statements or cross-reference `nature-data` |
| Figures/tables | Number and caption rules | Adjust in-text references |
| References | Limits or style | Flag only if writing must change |

## Output Rule

Always separate:

- `Official requirement`: directly supported by journal or publisher page
- `Inferred pattern`: observed in exemplar papers
- `Author decision needed`: cannot be resolved without the author's manuscript facts
